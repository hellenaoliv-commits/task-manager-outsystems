# Task Manager - OutSystems CRUD

Este projeto foi desenvolvido durante meus estudos em OutSystems, com o objetivo de praticar na prática como funciona um sistema completo de CRUD.

Criei um sistema de gestão de tarefas onde é possível listar, adicionar, editar e excluir tarefas. Na exclusão, implementei uma mensagem de confirmação para garantir mais segurança antes de apagar uma informação.

Durante esse projeto, consegui entender melhor como funciona a estrutura de dados, a lógica por trás das ações e como conectar tudo em uma aplicação funcional.

Esse foi um passo importante na minha evolução na área de desenvolvimento, saindo da teoria e realmente construindo algo que funciona.

## Funcionalidades

- Listagem de tarefas  
- Criação de novas tarefas  
- Edição de tarefas  
- Exclusão com confirmação de segurança

  ## O que eu pratiquei

- Estruturação de dados (entidades)  
- Implementação de operações CRUD  
- Lógica de programação  
- Navegação entre telas  
- Manipulação de formulários  
- Experiência do usuário (confirmação antes de excluir)
